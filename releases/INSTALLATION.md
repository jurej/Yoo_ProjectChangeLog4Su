# Installation Instructions

## Installing from .rbz File

1. **Download the .rbz file** for your desired version
2. **Open SketchUp**
3. **Go to Extensions Manager**:
   - Windows: `Window` → `Extension Manager`
   - macOS: `Window` → `Extension Manager`
4. **Click "Install Extension"** button
5. **Browse to the downloaded .rbz file** and select it
6. **Click "Install"**
7. **Restart SketchUp** if prompted

The plugin will now be available in the `Plugins` menu.

## Manual Installation (Alternative)

If you prefer manual installation:

1. **Rename the .rbz file to .zip**
2. **Extract the contents** to a temporary folder
3. **Copy the files** to your SketchUp Plugins folder:
   - Windows: `C:\Users\[YourUsername]\AppData\Roaming\SketchUp\SketchUp [Version]\SketchUp\Plugins\`
   - macOS: `~/Library/Application Support/SketchUp [Version]/SketchUp/Plugins/`
4. **Restart SketchUp**

## Available Versions

- **v1.3.0** - Latest version with skip threshold option (set to 0 to disable skipping)
